### Comandos do terminal Bash
1. **ls -a (List)** = Lista o conteúdo do diretório atual 
2. **cd (Change Directory)** = Muda o diretório de trabalho atual  para outro diretório
3. **pwd (Print Work Directory)** = Retorna o caminho absoluto do diretório de trabalho atual 
4. **mkdir (Make Directory)** = Cria um novo diretório no diretório de trabalho atual
5. **cat (Concatenate file)** = Retorna o conteúdo de um arquivo de texto, a partir da concatenção das suas linhas 
6. **rm (Remove)** = Exclui um arquivo ou diretório.
7. **clear** = Limpa o conteúdo do terminal
8. **nano** = Abre um arquivo de texto com o editor de texto nano
